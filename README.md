# Hangi Nişanyan'sın?

Twitter'da gündem olan **Nişanyan ailesi** kavgasından ilham alan, tamamen eğlence amaçlı bir karakter testi. 12 soruda Sevan, Arsen, Tavit veya Ira'dan hangisine benzediğini söyler.

- **Tek dosya:** `index.html` — saf HTML/CSS/JS, framework yok
- **Paylaşılabilir sonuç kartı:** ekran dışı düğüm → `html2canvas` ile birebir görsel (kopyala / indir / mobil paylaş)
- **Anonim global sayaç + sonuç dağılımı:** [abacus](https://abacus.jasoncameron.dev) (kişisel veri toplanmaz)
- **Yazı tipleri:** Space Mono + Inter (Google Fonts)

## Çalıştırma

Statik site; bir HTTP sunucusu yeterli (görsel üretimi `file://`'de çalışmaz):

```bash
python3 -m http.server 8080
# http://localhost:8080
```
